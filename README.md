# regression_analysis

This repo contains code for the regression analysis course I took in smester 2 2021 using `R` and `python`. 

This was forked from my teachers repo, yet I have changed some of the code to solidify my understanding. 

It covers a very general introduction to econometrics. 

## The content includes:

- Regression Estimation (OLS) & Inference
- Endogeneity in Instrumental Variables (IV)
- Time Series Modelling (static)
- Panel Data (longitudinal data) Modelling
- Treatment Effects via Difference-in-differnces (DiD)
- Maximum Likelihood Estimation and Numerical Methods
- Classification Problems
