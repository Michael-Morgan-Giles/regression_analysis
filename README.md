# regression_analysis

This repo contains code for the regression analysis course I took in 2021S2 using `R` and `python`. 

This was forked from my teachers repo, yet I have changed some of this code to solidify my understanding. 

It covers a very general introduction to econometrics. 

Whilst a basic understanding of linear algebra and calculus would be helpful, it isn't necessary to follow along.

## The content includes:

- Regression Estimation (OLS) & Inference
- Endogeneity in Instrumental Variables (IV)
- Time Series Modelling
- Panel Data (longitudinal data) Modelling
- Treatment Effects
- Maximum Likelihood Estimation and Numerical Methods
- Classification Problems



- W1 : Regression estimation
- W2 : Post Regression inference
- W3 : Generalization of Regression
- W4 : Endogeneity and Instrumental Variable 
- W5 : Simultaneous Equations and multi-stage Regression
- W6 : Time Series modelling
- W7 : Panel Data (longitudinal data) Modelling
- W8 : Treatment effects
- W9 : Maximum Likelihood Estimation and Numerical Methods
- W10 : Classification (aka discrete choice modelling)
- W11 : Generalization of Classification : linear models with limited dependent variables
- W12 : Generalization of Classification :  Machine Learning algorithms
- A1, A2: Assignment 1 and 2
- Q1, Q2, etc : illustration of GLM
